# atariML
Simple atari machine learning exercise

## Installation

```
git clone https://github.com/JustSomeDude2001/atariML.git

pip install --upgrade pip

pip install tensorflow
pip install gym[atari]
```

## Usage

To play breakout:

```
python breakout.py
```
